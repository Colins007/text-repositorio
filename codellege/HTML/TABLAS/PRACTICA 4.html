Marcos de grabajo de java

La palabra inglesa "framework"(marco de trabajo) define, en terminos generales un conjunto estandarizado de comceptos, practicas y criterios para enfocar un tipo de problematica particular que sirve como referencia, para enfrentar y resolver nuevos problemas de indole similar.

En el desarrolo de sofware, un framwork o infrstructura dijital, es una estructura conceptual y tecnologica de soporte definido, normalmente con artefactos o modulos de sofware concretos, que puede servir de base para la organizacion y desarollo de sofware. Tipicamente hablando, puede incluir soporte de programas bibliotecas y un lenguaje interpretado, entre otras herramientas, para asi ayudar a desarrollar y unir componentes de un proyecto.
