Ejemplo de APPLEt Java

Un applet conforme a lo que encontramos en las paginas de oracle: "es una aplicacion hecha en java que se llama desde un documento HTML y se ejecuta dentro de un navegador". Es necesario recordar que el navegador debe tener instalado el plugin de la Maquina Virtual de Java(JVM)

Maquina virtual de Java: Es el programa que se encarga de ejecuta el bytecode generado por el compilador de Java

Para entender la forma en que se maneja los aplets vamos a ver un ejemplo que presenta el clasico Hola mundo


      impor java.awt.Graphics;
      impor javax.swing.JApplet;
      public class HolaMundo extends JApplet {
      public void paint(Graphics g) {
      super.paint(g);
      g.drawString("Hola mundo!!",10,20);
      }
      }

¡Como puede observarse la única variable que uiza ete applet es g que representa un objeto de la case Graphics sobre el cual vamos a dibujar la cadena de texto que resua de ejecutar el apple

Para ejecutarlo es necesario mandarlo lamar desde una página HTML utilizando la etiqueta "applet" o la etiqueta "object". En el primer caso el archivo class se declara en el atributo code,en el segundo se declara un elemento param donde sus atributos name será code y value será el nombre del applet Cabe mencionar que en el segundo caso es necesario declarar otros elementos param adicionales y en elemento object declarar algunos parámetros como clossid, codebase y type. En ambos casos es necesario declarar el tamaño de la ventana del applet con los Parámetros widlh y height


La salida del programa ser Hola mundo!
